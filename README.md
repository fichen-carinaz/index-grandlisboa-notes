# index-grandlisboa-notes

本仓库用于归档和发布多个独立的 HTML 页面。所有页面均为静态内容，不依赖后端服务，可直接通过浏览器访问。

## 项目简介

这是一个非结构化的页面归档仓库，主要存放一些零散的 HTML 文件。这些页面内容彼此独立，不针对特定域名或网站，仅作为资料的整理与备份。

## 目录说明

```
.
├── README.md          # 本说明文件
├── index.html         # 仓库入口页面（如有）
└── pages/             # 归档的 HTML 页面存放目录
    ├── example1.html
    ├── example2.html
    └── ...
```

- `pages/` 目录下存放所有归档的 HTML 文件，文件名及内容因归档批次而异。
- 根目录下的 `index.html` 为可选入口页面，非必需。

## 页面归档说明

- 所有页面均为独立的 HTML 文档，无跨页面依赖。
- 页面可能包含静态资源（如 CSS、JS、图片），资源通常以内联或相对路径方式引用。
- 归档内容不保证时效性、完整性或准确性，仅作记录用途。

## 维护说明

- 本仓库不定期更新，新增或修改页面。
- 不承诺任何形式的持续维护或更新频率。
- 欢迎通过 Issue 或 Pull Request 提交修正建议，但请注意仓库定位为个人归档，不一定接受外部贡献。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接在浏览器中打开所需的 HTML 文件。
3. 也可通过 GitHub Pages 等静态托管服务部署后访问（需自行配置）。
